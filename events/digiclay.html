DigiClay

Model and animate an original 3D sequence in Blender — originality, and technical skills all factor into the score.

Software: Blender
Mode: Individual / Duo
Topic: Revealed one day prior to the event on the Notice Pannel
Overview

Participants create an original 3D animation using Blender. Topic is given one day prior to the event. All models must be original — pre-built assets are not allowed.

Judging Criteria
  • Originality — unique models and concept.
  • Technical Skills — rigging, animation, and scene composition.

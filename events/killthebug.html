Kill The Bug

15 Java snippets riddled with errors — 1 hour to hunt and fix every bug, on paper. — 2 hours to hunt and squash every bug.

Language: Java
Mode: Individual, Offline, Paper Based
Questions: 15
Duration: 2 Hours
Categories: Junior (9–10) & Senior (11–12)
Overview

Participants receive a printed sheet of 15 buggy Java snippets and must identify and correct all errors directly on the answer sheet within 1 hour. This is a single open category — Classes 9 to 12 all compete together. No computer is involved.

Error Types
  • Syntax errors — incorrect Java grammar or structure.
  • Runtime errors — code that crashes during execution.
  • Logic errors — code that runs but produces wrong output.
Grading
  • Marks awarded for each correct fix on the answer sheet.
  • All three error types — syntax, runtime, and logic — are individually marked.
  • Partial credit given where applicable.
  • Full points awarded if the corrected code runs without errors.

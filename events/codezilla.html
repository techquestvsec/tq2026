Coding

Codezilla

Individual Java programming —— solve as many questions as you can from a 25-question set, judged on functionality and trick questions.

Language: Java
Mode: Individual
Questions: 25 per set(2 seperate sets- one per category)
Duration: 3 hours
Format: Computer-based
Max Participants: 8 per category
Categories: Junior (9–10) & Senior (11–12)
Overview

Participants individually develop programs in Java. The event is open to Classes 9–12, split into Junior and Senior categories. Each category receives its own question set tailored to their level. Participants attempt as many of the 25 questions as they can within the 3-hour time limit. File naming format: NameQuestionNumber (e.g. xyz2)

Pre-Event Selection Round
SELECTION ROUND — 2 WEEKS BEFORE EVENT
Registration & Selection
After registration, all candidates are called for a selection round held 2 weeks prior to the main event. Participants are given 3 questions as per their class level and must solve as many as possible within 15 minutes. Candidates are ranked by speed and whether their code works correctly. Only selected candidates proceed to the main event.
Event Day
3 hours — MAIN EVENT
Coding Session
Participants receive their question set (25 questions) on the day of the event. Two separate sets are used — one for each category — so questions match your level. Attempt as many as you can, save each file as NameQuestionNumber, and submit all code files to the event head at the end of the session.
Judging Criteria
  • Functionality — does the program work correctly?
  • Trick Questions —are edge cases and trick questions handled?

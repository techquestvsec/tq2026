Lead The Stick

Animate stick figures in Pivot — fluid movement, compelling narrative, and technical precision are what the judges look for.

Software: Pivot Animator v5
Mode: Individual
Categories: Junior(class 5-8) | Senior(class 9-10)
Max Participants: 10 Participants per Category
Duration: 2 hours(total event time)
Format: Computer Based
Rounds: one
Venue: Computer Lab
Overview

Participants create stick-figure animations using Pivot Animator. The topic is given on the day of the event. Style and length are open-ended — judges focus on movement quality and storytelling.

Selection Round

It will take place two weeks before the main event. Participants will have to complete a given task using a given time using tools in the software. Duration will be 2 hours. Judging critera will be creativity+ smoothness of animation and storyline. Top 20 participants will proceed.

Event Structure

FINALE ROUND Name : Final Showdown Duration : 2 hours Description : A creative 2D animation event using Pivot Animator where participants bring stick figures to life through short, imaginative animations as per the topic. Format : Computer

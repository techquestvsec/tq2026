From The Scratch

Build a game or animated story in Scratch 2 / Scratch 3 — logic, creativity, and code quality are all judged.

Platform: Scratch 2 / Scratch 3
Mode: Individual
Categories: Junior (5–8) & Senior (9–12)
Overview

Participants create an algorithm, game, or animated story using Scratch. The project topic is given on the day of the event. Selection for the event is based on a pre-elimination test.

What to Submit
  • A complete Scratch project (game or animated story).
  • Must be created entirely during the event — no pre-built projects.
Judging Criteria
  • Creativity — originality of concept and execution.
  • Code Quality — efficient use of Scratch blocks and logic.
  • Presentation — visual polish and user experience.
